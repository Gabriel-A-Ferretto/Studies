## Tipos de dados existentes em C++



| Tipo de Dado  |  Bits | Faixa de Valores |
|------|-------|--------|
| **char** | **8** |**-127 a 127**  |
| **bool** | **8** | **true e false** |
| **int** | **32** | **-2.147.438.647 a 2.147.483.647** |
| **float** | **32** | **7 digitos significativos** |
| **double** | **64** | **dígitos significativos** |
